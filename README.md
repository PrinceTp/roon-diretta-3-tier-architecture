# Three-tier Roon + Diretta audio chain with network isolation for low-noise, high-fidelity playback.

A documented setup for building a **high-performance Diretta-based audio system** using a three-tier architecture: **Roon Core → Diretta Host → Diretta Target → DAC**.

This approach isolates processing, transmission, and playback to reduce electrical noise and improve audio quality.

---

## Architecture

```
Roon Core → Network → Diretta Host → Diretta Target → DAC
```
